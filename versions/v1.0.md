# BRT 1.0

Release date: July 2026

## Overview

BRT (Bulgarian Reader Test) 1.0 is the first public version of a community methodology for evaluating the accessibility of web content using browser reading features.

The methodology focuses on whether the main content of a web page remains accessible and readable through Reader Mode / Reader View.

## Included

- Introduction and methodology goals
- Core principles
- Reader Mode explanation
- Google Chrome instructions
- Mozilla Firefox instructions
- Required tools
- Testing procedure
- PASS / PARTIAL / FAIL evaluation criteria
- First practical test
- Initial project documentation
