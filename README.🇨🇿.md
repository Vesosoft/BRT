# BRT – Bulgarian Reader Test

## Test dostupnosti webového obsahu

## Webová stránka

Otevřete stránku projektu BRT:

`index.html`

BRT (Bulgarian Reader Test) je veřejná neoficiální metodika pro rychlou kontrolu toho, zda je webová stránka pohodlná ke čtení pomocí funkcí přístupnosti v prohlížeči.

Cílem je umožnit každému člověku, bez nutnosti technických znalostí, ověřit, zda je hlavní obsah webové stránky dostupný.

## Proč je BRT potřeba?

Mnoho lidí používá pomocné funkce, například:

- režim čtení (Reader Mode / Reader View);
- zvětšení textu;
- úpravu kontrastu;
- tmavý režim;
- další nastavení pro snadnější čtení.

Pokud webová stránka s těmito funkcemi nepracuje správně, někteří uživatelé mohou ztratit přístup k jejímu obsahu.

## Co BRT kontroluje?

BRT kontroluje:

- zda se správně načítá hlavní text;
- zda se zobrazuje celý článek;
- zda jsou odstraněny rušivé prvky;
- zda lze text pohodlně číst;
- zda existují problémy s obrázky a dalším obsahem.

## Reálné testovací případy

BRT obsahuje zdokumentované testy přístupnosti provedené na skutečných webových stránkách.

Příklady:

- Test #001 - Meteo Balkans  
  Reader Mode lze aktivovat, ale hlavní obsah se neextrahuje.

- Test #002 - Miele  
  Chrome Reader Mode funguje. Firefox Reader View není k dispozici.

## Princip

BRT není nástroj pro kritiku webových stránek.

Je to nástroj pro hledání překážek pro čtenáře a pro zlepšení přístupnosti webového obsahu.
